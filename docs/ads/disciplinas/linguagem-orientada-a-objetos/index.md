# 🧩 Linguagem Orientada a Objetos (LOO)

## 📌 O que é?
A Linguagem Orientada a Objetos é um paradigma de programação baseado na organização do código em objetos.

---

## ❗Importância
A importância se dá pela necessidade da LOO para o desenvolvimento de sistemas e aplicativos modernos. Antes do O.O, os códigos eram sequenciais e a programação orientada a objetos trouxe a possibilidade de agrupar dados, reutilizar códigos e realizar manutenções com um comando, o que ajudou com os sistemas mais complexos.

---

## 1. Linguagens de Programação OO

- Java: A mais popular quando se trata de orientação a objetos. Foi criada inicialmente com o principio de "escrever uma vez e rodar em qualquer lugar" (WORA), ou seja, o código Java pode ser executado em diferentes plataformas. Sendo essa uma linguagem puramente orientada a objetos.
- Python: Linguagem muito popular entre na área de Machine Learning, versatil, de fácil aprendizado e que permite programação orientada a objetos mesmo não sendo puramente orientada a objetos.
- C++: Baseada na linguagem C, adiciona suporte a POO e é muito utilizada no desenvolvimento de jogos e softwares.

## 2. Ambiente de Desenvolvimento Integrado (IDE)
- Eclipse: Muito utilizado para o desenvolvimento em Java, com suporte para a criação de interfaces gráficas e ferramentas de depuração e controle de versão.
- PyCharm: um IDE muito utilizado para Python, com funcionalidades que suportam a programação orientada a objetos, facilitando a organização e visualização dos métodos e atributos.
- NetBeans: um IDE que suporta diversas linguagens, mas com uma forte orientação para Java.

## 3. Frameworks
  Os frameworks fornecem um conjunto de ferramentas, bibliotecas e práticas recomendadas que facilitam o desenvolvimento de aplicações. Ajudam a simplificar tarefas fornecendo um bloco de códigos contendo uma estrutura pronta, assim facilitando a manutenção do código. São exemplos de frameworks:

- Spring (Java): um dos frameworks mais conhecidos para Java, o Spring simplifica o desenvolvimento de aplicações empresariais e promove o uso de componentes modulares.
- Django (Python): um framework que facilita o desenvolvimento de aplicações web. Com o Django, o desenvolvedor cria classes que representam modelos de dados e controla seu acesso e visibilidade.
- Qt (C++): um framework multiplataforma para desenvolvimento de interfaces gráficas. Com o Qt é possível criar interfaces complexas e interativas.
