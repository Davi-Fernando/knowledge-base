# 🧩 Linguagem Orientada a Objetos (LOO)

## 📌 O que é?
A Linguagem Orientada a Objetos é um paradigma de programação baseado na organização do código em objetos.

---

## ❗Importância
A importância se dá pela necessidade da LOO para o desenvolvimento de sistemas e aplicativos modernos. Antes do O.O, os códigos eram sequenciais e a programação orientada a objetos trouxe a possibilidade de agrupar dados, reutilizar códigos e realizar manutenções com um comando, o que ajudou com os sistemas mais complexos.

---

## 😁 Vantagem:
- Modularidade
- Escalabilidade do Sistema
- Abstrair conceitos do mundo real

## 😒 Desvantagem:
- Maior custo de aprendizado
- Complexidade em padrões de design

---

## 💡 Insight
- Facilita manutenção
- Aproxima código do mundo real
